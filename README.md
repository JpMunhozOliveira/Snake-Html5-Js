## Objective

The objective of the Snake game is to achieve the highest possible score by skillfully maneuvering the snake through the game area and avoiding collisions. As the snake eats the food, the player's score increases. However, if the snake collides with the game's boundaries or its own body, the game ends. The challenge lies in strategically navigating the snake, seeking out food while avoiding obstacles, in order to increase the score and set new records.
